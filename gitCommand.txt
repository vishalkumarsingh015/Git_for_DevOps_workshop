- git branch – List all branches
- git branch <name> – Create a new branch
- git checkout <branch> – Switch to a branch
- git checkout -b <branch> – Create and switch to a new branch
- git merge <branch> – Merge a branch into the current branch
- git rebase <branch> – Reapply commits on top of another base tip

Remote Repositories

- git remote -v – List all remotes
- git remote add <name> <url> – Add a new remote
- git fetch <remote> – Fetch updates from a remote repo
- git pull – Fetch and merge changes from the remote repo
- git push – Push changes to the remote repo
- git push <remote> <branch> – Push a specific branch to remote

Undoing and Resetting

- git reset <file> – Unstage a file
- git reset --hard – Reset working directory and index to last commit
- git checkout -- <file> – Discard changes in a file
- git revert <commit> – Revert a specific commit

Stashing

- git stash – Stash changes in a dirty working directory
- git stash apply – Apply stashed changes
- git stash pop – Apply and remove the latest stash

Tagging

- git tag – List all tags
- git tag <name> – Create a tag
- git tag -d <name> – Delete a tag
- git push --tags – Push tags to remote

Other Useful Commands

- git show <object> – Show details about an object (commit, tag, etc.)
- git blame <file> – Show who changed what and when in a file
- git config --global user.name "Your Name" – Set global username
- git config --global user.email "your@email.com" – Set global email

Help

- git help <command> – Show help for a command
- git <command> --help – Another way to get help
